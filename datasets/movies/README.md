## Fields

| Field | Type | Description |
|-------|------|-------------|
| name | string | Movie title |
| slug | string | URL-safe identifier |
| rating | float | Average user rating (0-10) |
| vote_count | int | Number of votes |
| release_date | date | Theatrical release date |
| language | string | Original language (ISO code) |
| genres | string | Genre list |
| runtime_min | int | Runtime in minutes |
