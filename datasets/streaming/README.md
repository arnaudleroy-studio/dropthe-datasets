# Streaming Availability by Service and Country

1,938 aggregated records showing title counts per streaming service, per country, per availability type.

Derived from 685,896 individual streaming records across all major platforms.

## Source

[DropThe.org](https://dropthe.org) — Data platform tracking streaming availability across 20+ services in 50+ countries.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| service_name | string | Streaming service (Netflix, Disney+, etc) |
| country_code | string | ISO country code |
| availability_type | string | subscription, rent, buy, free |
| title_count | int | Total availability records |
| unique_titles | int | Unique titles available |

## Coverage

- 685,896 individual streaming records
- 20+ streaming services
- 50+ countries
- Updated: February 2026
