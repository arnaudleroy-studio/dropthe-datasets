# Cryptocurrency Mortality Index

15,261 cryptocurrencies tracked with current status, market data, and mortality indicators.

## Source

[DropThe.org](https://dropthe.org) — Data platform tracking 1.83M entities including 16K+ cryptocurrencies.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| name | string | Cryptocurrency name |
| symbol | string | Ticker symbol |
| slug | string | URL-safe identifier |
| status | string | Current status (active, dead, inactive) |
| market_cap | float | Current market capitalization (USD) |
| price | float | Current price (USD) |
| volume_24h | float | 24-hour trading volume (USD) |
| circulating_supply | float | Coins in circulation |
| max_supply | float | Maximum supply cap |
| launch_date | date | Date first tracked |
| last_updated | datetime | Last data refresh |
